aniketbhatnagar.com
======================
My personal website

## Run locally
- [Install hugo](https://gohugo.io/installation/)
- cd src && hugo server

## Publish
1. Build using ./bin/build.sh
2. Commit and push changes
3. A [github deployment](https://github.com/aniketbhatnagar/aniketbhatnagar.github.io/deployments) should trigger
